# Workplace Internal Automation & Drawing Indexer (Matelfe Gest / CAD Vault)
*Internal Production & Workflow Optimization Tool*

## Project Overview
Developed to eliminate inefficiencies in managing large repositories of technical drawings, manufacturing specifications, and commercial proposals. This application replaces rigid, manual directory browsing with an automated, data-driven indexer.

## Technical Stack & Features

### 1. Data Architecture (SQL)
* Designed a relational SQL database schema to index engineering metadata from CAD drawings (`.dwg`, `.dxf`) and active manufacturing proposals.
* Centralized technical documentation tracking, eliminating data redundancy and reducing revision mismatch errors between drafting and production stages.

### 2. Application Logic & UI (Python & Streamlit)
* Built using **Python** with the **Streamlit** framework to deliver a lightweight, responsive internal web interface.
* Implemented fast metadata query algorithms allowing production teams to instantly retrieve specific electrical schematics, transformer station configurations, and revision histories.
* Designed for low-maintenance deployment, running efficiently on local infrastructure without frontend overhead.

### 3. Workflow Impact
* Dramatically minimized the drafting-to-production turnaround time.
* Standardized document indexing across the team, reducing search friction and accelerating project licensing validation.
